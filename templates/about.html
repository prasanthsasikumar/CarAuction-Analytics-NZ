About This Project

Comprehensive vehicle auction data from Manheim New Zealand

What is CarAuction Analytics?

CarAuction Analytics NZ is a comprehensive data platform that tracks vehicle auction listings from Manheim New Zealand. The project automatically scrapes auction data daily, providing historical trends, price analytics, and insights into the used vehicle market.

This platform serves researchers, data analysts, vehicle buyers, and anyone interested in understanding the New Zealand vehicle auction market.

Data Fields

Each record includes: Manufacturer, Model, Year, Mileage, Price, Damage information, Location, Registration status, Body type, Transmission, Fuel type, Engine size, Seats, Keys, and more.

How It Works

  1. Daily Scraping: Automated scraper runs daily to collect all vehicle listings from Manheim NZ
  2. Data Processing: Raw data is cleaned, deduplicated, and enriched with derived features
  3. Storage: Data is stored in multiple formats (CSV, Parquet, SQLite) for different use cases
  4. Visualization: Interactive dashboards provide insights into trends and patterns
  5. API Access: RESTful API enables programmatic access to the data

Data Quality & Ethics

This project follows ethical web scraping practices:

  • Publicly available data only
  • Respectful scraping rate (once per day)
  • No personal information collected
  • Data cleaning to remove duplicates and errors
  • Transparent methodology and documentation
Note: This data is for informational and research purposes only. Prices and availability should be verified directly with Manheim NZ.
